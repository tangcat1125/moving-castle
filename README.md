# 3D moving-castle：沙漠防線 (Moving Castle 3D)

這是一個以 Three.js 打造的 3D moving-castle塔防與即時戰略 RPG 遊戲。城堡行駛在無垠的沙漠中，遭遇隨機來襲的可愛二頭身殭屍，玩家必須架設防線並召喚英靈出城對戰。

## 🎮 操作指令說明

### 1. 城堡移動
* 使用 **方向鍵 ◀ ▶** 或 **A / D**：旋轉城堡車頭（左右轉向）。
* 使用 **方向鍵 ▲ ▼** 或 **W / S**：朝城堡面對的方向行駛（前進 / 後退）。
* **相機**：會保持在城堡正後方，隨著您的轉向而自動平滑盤旋跟隨。

### 2. 架設防禦塔 (最多 4 隻)
* 點擊下方終端機「🏹 弓箭手」、「🪓 擲斧者」、「⚙️ 連弩車」或「💣 擲彈兵」按鈕，城堡將**自動尋找空平台即時架設並開火**。
* 炸彈降落時會產生**橘紅色半透明 AoE 擴散爆炸**，對範圍內多個殭屍造成傷害。

### 3. 召喚英雄出城
* 點擊「💂 召喚小兵」、「🛡️ 召喚將軍」或「👑 👑 姬公主」，他們會從城堡中門出生，自動前去攔截殭屍：
  * **姬公主 (快速攻擊)**：砍滿 10 次後，施展**火焰劍**朝前方直線瞬移，秒殺沿路殭屍。
  * **將軍 (高血坦克)**：砍滿 5 次後，將殭屍**抓起至半空進行地球摔重擊**，引發大範圍波震。
  * **姬公主 (治療環)**：每 2 秒發送治癒環治癒城堡與周邊小兵。

### 4. 擊殺獎勵
* 每次射殺、砍死殭屍都會獲得黃金（HUD 點數），可用於召喚更多守衛與英雄！
